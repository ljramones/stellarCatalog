# TRIPS for Windows 10/11

TRIPS installer for Windows 10 and Windows 11 on x86-64 processors.

## Current Release

No installer is currently available for this platform. Check back for future releases.

## Installation Instructions (when available)

1. Download the installer (`.exe` or `.msi`) from this directory.
2. Double-click the installer to launch the setup wizard.
3. Follow the on-screen prompts to complete installation.
4. Windows may display a SmartScreen warning since the app is not signed with a recognized certificate. Click **More info** > **Run anyway** to proceed.

## Uninstallation

Go to **Settings > Apps > Installed apps**, find **TRIPS**, and click **Uninstall**.
