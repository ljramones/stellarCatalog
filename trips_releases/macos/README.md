# TRIPS for macOS

Platform installers for TRIPS on macOS, organized by processor architecture.

## Directories

- **[appleSilicon/](appleSilicon/)** - For Macs with Apple Silicon (M1, M2, M3, M4) processors
- **[x64_intel/](x64_intel/)** - For Macs with Intel x86-64 processors

## Identifying Your Mac's Processor

1. Click the Apple menu () > **About This Mac**.
2. Look for the **Chip** or **Processor** field:
   - If it says **Apple M1**, **M2**, **M3**, or **M4** -> use `appleSilicon/`
   - If it says **Intel** -> use `x64_intel/`
