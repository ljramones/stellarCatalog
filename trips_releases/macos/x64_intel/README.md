# TRIPS for macOS - Intel x86-64

TRIPS installer for macOS on Intel x86-64 processors.

## Current Release

No installer is currently available for this platform. Check back for future releases.

## Installation Instructions (when available)

1. Download the `.dmg` file from this directory.
2. Double-click the `.dmg` file to mount it.
3. Drag the **TRIPS** application into your **Applications** folder.
4. Eject the disk image.
5. On first launch, you may need to right-click the app and select **Open** (or go to **System Settings > Privacy & Security** and click **Open Anyway**) since the app is not signed with an Apple Developer certificate.

## Uninstallation

Drag **TRIPS** from your Applications folder to the Trash.
